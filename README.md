# Cyber-Shield-Project
​"A Python-based security simulation tool featuring user authentication, access control levels, and a network vulnerability scanner."
# 🛡️ Cyber-Shield v1.0

A professional Python security script designed to demonstrate the fundamentals of **Cybersecurity** and **Access Control Systems**.

## 🚀 Key Features
- **User Authentication:** Validates users against a secure database.
- **Security Levels:** Assigns specific permissions based on user roles (Admin/Guest).
- **Blacklist & Quarantine:** Automatically detects and isolates suspicious login attempts.
- **Network Scanner Simulation:** Simulates a vulnerability scan on suspicious IP addresses using the `time` module for realism.
- **Input Validation:** Enforces strong password policies and age-based restrictions.

## 🛠️ Technologies Used
- **Python 3.x**
- **Modules:** `time` (for process simulation)

## 📖 How it Works
1. The system checks if the user is in the **Blacklist**.
2. It validates the password length to ensure security.
3. Depending on the user role, it grants access to the **Vulnerability Scanner**.
4. All suspicious activities are logged into a **Quarantine List**.
